# Install and create db, user and set password

## Install MariaDB server

```
./mariadb-install.sh
```
## For basic config

```
./mariadb-create.sh database user password
```
## For backup

```
./mariadb-backup.sh
```

## For restore backup

```
./mariadb-restore.sh
```
